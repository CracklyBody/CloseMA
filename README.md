# CloseMA
OpenGL first person shooter, update come soon

### **Key bindings:** ###

**W** = move forward

**A** = move backward

**S** = move left

**D** = move right

**MOUSEWHEEL** = zoom

**SPACE** = take lamp (shadows)

**F** = flashlight
